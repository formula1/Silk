NetworkInstance

new NetworkInstance(nethost, identity)

A WebRTC connection between you and another party. Extends message duplex allowing you to add and send get, trigger and pipe requests to the other party. This is an implementation of https://developer.mozilla.org/en-US/docs/Web/Guide/API/WebRTC
