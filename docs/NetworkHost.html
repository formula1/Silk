Class: NetworkHost

NetworkHost

new NetworkHost(url, info, config, sconfig)

Creates a new Network host. A network host provides a manner to find users with a compatible app as yours. The manner in which they find other users is meaningless. However, it provides a way to make an offer to another user and to accept an offer.
Parameters:
Name Type Description
url string the url that hosts the users you wish to find
info object the info you wish to give inorder for other users to find you
config object configuration, mostly just specifying ice servers
sconfig object data configuration, in case you are interested in that
